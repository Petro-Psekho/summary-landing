# Interactive Resume SPA

Одностраничный интерактивный сайт-резюме Петра Псехо.

## Запуск

1. Откройте `index.html` в браузере.
2. Для корректной локальной загрузки можно использовать любой static server.

## Файлы

- `index.html` — структура и контент резюме
- `style.css` — современный UI, адаптив, анимации
- `app.js` — интерактив: частицы, reveal-эффекты, счетчики, параллакс, прогресс скролла
